# Affice

## Brief
    Приложение представляет собой http-сервер с одним хендлером.
    Хендлер на вход получает POST-запрос со списком url в json-формате
    Сервер запрашивает данные по всем этим url и возвращает результат клиенту в json-формате



## Requirements
1. Eсли в процессе обработки хотя бы одного из url получена ошибка, обработка всего списка прекращается и клиенту возвращается текстовая ошибка 
2. Для реализации задачи следует использовать Go 1.13 или выше
3. Использовать можно только компоненты стандартной библиотеки Go
4. Сервер не принимает запрос если количество url в в нем больше 20
5. Сервер не обслуживает больше чем 100 одновременных входящих http-запросов
6. Для каждого входящего запроса должно быть не больше 4 одновременных исходящих
7. Таймаут на запрос одного url - секунда
8. Обработка запроса может быть отменена клиентом в любой момент, это должно повлечь за собой остановку всех операций связанных с этим запросом
9. Сервис должен поддерживать 'graceful shutdown'
10 Результат должен быть выложен на github
